## 🚀 如何使用？

只需将某个播放列表的链接粘贴到**支持直播流的播放器**中，然后点击打开即可。 ([GitHub][1])


* 本仓库**不存储任何视频文件**

* 项目认为这些内容是版权方**主动公开提供的**

如果你是版权方并认为某些链接侵权：

* 可以提交 PR 或 issue 请求移除
* 但请注意：

  * 删除链接 ≠ 删除源内容
  * 本仓库**不控制视频源内容**
  * 实际内容托管在第三方网站

👉 若需彻底删除内容，应联系**实际托管视频的服务器提供方**，而不是 GitHub 或该仓库维护者。 ([GitHub][1])

# ✅ 总结（给你一个技术视角理解）

这个项目本质上是：

> 一个**标准化 IPTV 聚合索引层**

核心价值：

* 用 **M3U** 做统一分发格式
* 将“频道数据 / 播放地址 / EPG / API”拆分成多个仓库
* 支持播放器（VLC / Kodi / Plex 等）直接消费

👉 简单说：
它不是“电视服务”，而是**“电视流的索引 + 数据基础设施”**
