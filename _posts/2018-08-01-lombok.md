# lombok

大幅度缩减代码的编写量。

# @Builder 与序列化的冲突

有时候使用 `@Builder`，此注解默认会使得无参构造器私有化。

然而 json 反序列化的时候，需要无参构造器。

如下使用即可

```java
@Data
@Builder
public class OutNotifyCbMessage implements Serializable {

    @Tolerate
    public OutNotifyCbMessage() {
    }
}
```
