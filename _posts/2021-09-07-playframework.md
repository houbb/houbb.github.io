# playframework

Play 框架结合了生产力和性能，可以轻松地使用 Java 和 Scala 构建可扩展的 Web 应用程序。 

Play 对开发人员友好，具有“只需点击刷新”的工作流程和内置的测试支持。 

借助 Play，应用程序由于无状态和非阻塞架构而可预测地扩展。

默认情况下是 RESTful，包括资产编译器、JSON 和 WebSocket 支持，Play 非常适合现代 Web 和移动应用程序。
