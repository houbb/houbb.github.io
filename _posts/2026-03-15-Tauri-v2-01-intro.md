# 开始使用（Start）

## Tauri 2.0

创建 **小型、快速、安全的跨平台应用程序**。

---

## 创建项目（Create a Project）

### Bash

```
sh <(curl https://create.tauri.app/sh)
```

### PowerShell

```
irm https://create.tauri.app/ps | iex
```

### Fish

```
sh (curl -sSL https://create.tauri.app/sh | psub)
```

### npm

```
npm create tauri-app@latest
```

### Yarn

```
yarn create tauri-app
```

### pnpm

```
pnpm create tauri-app
```

### Deno

```
deno run -A npm:create-tauri-app
```

### Bun

```
bun create tauri-app
```

### Cargo

```
cargo install create-tauri-app --locked

cargo create-tauri-app
```

# 核心特点

## Frontend Independent（前端框架无关）

你可以将 **现有的 Web 技术栈** 直接用于 Tauri，
或者基于它开始一个新的项目。

Tauri 支持 **任何前端框架**，因此你无需改变自己的技术栈。

---

## Cross Platform（跨平台）

你可以使用 **同一套代码库** 构建应用程序，目标平台包括：

* Linux
* macOS
* Windows
* Android
* iOS

开发方式通常是：

* **前端**：JavaScript / HTML / CSS
* **应用逻辑**：Rust
* **系统集成**：Swift（iOS）或 Kotlin（Android）

---

## Maximum Security（最大化安全性）

安全性是 Tauri 团队最重要的优先级之一，
并驱动着该项目的大量设计和创新。

---

## Minimal Size（最小体积）

Tauri 使用 **操作系统原生 WebView 渲染器**，
因此应用程序体积可以 **小到约 600KB**。 ([Tauri][1])
