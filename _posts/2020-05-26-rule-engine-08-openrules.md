# 快速介绍

OpenRules® Decision Manager 允许业务人员开发、测试和维护基于规则的业务决策模型，并轻松部署在云端或本地。主要组件包括：

- 规则引擎：以最佳性能和可伸缩性执行业务规则
- 集成决策建模环境：允许业务用户构建、测试、调试、分析和部署决策模型
- 规则学习器：从历史数据中提取业务规则
- 规则求解器：解决决策优化问题
- 规则数据库：与关系型数据库通信
- 规则对话框：创建基于规则的 Web 问卷。

OpenRules® 面向与软件开发人员协作的主题专家（业务分析人员）。OpenRules® 不强迫其客户学习专有的图形界面或特殊的规则语言。他们可以使用熟悉的 MS Excel® 或 Google Sheets® 和标准文件管理器来编辑和执行他们的规则。OpenRules 配备了一个集成的决策建模环境“图形浏览器”，它会自动构建决策图，包括一个功能强大的图形调试器，能够逐一执行规则以简化复杂决策模型的分析。

使用 OpenRules® 创建和测试的决策模型不会对您的决策业务应用程序施加任何限制。它们可以自然地部署为容器化的决策服务，使用 AWS Lambdas、MS Azure Functions、RESTful Web Services、Spark、Docker 和 Kubernetes 在任何流行的云框架上。

OpenRules® Decision Manager 价格合理，完全符合现代无服务器架构的要求。它很适合在任何云容器、本地甚至智能手机上使用。
