# Docker 

回顾 docker 的学习之路，漫长无涯。

从 2016 年到今天，断断续续学过多次，但都没有任何长进。

时间一长不使用，就全然忘记。

## 基本原理

总结起来，原因很简单，底层原理没有掌握：

（1）Linux

（2）GoLang 协程等语言优势

（3）微服务相关

这些后期都将进行系统的学习和整理。
