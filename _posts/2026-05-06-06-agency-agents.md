# 🎭 The Agency：随时改变您工作流的 AI 专家团

一个完整的 AI 机构触手可及——从前端奇才到 Reddit 社群忍者，从趣味注入者到现实核查者。每个 agent 都是拥有独特个性、流程和可靠交付物的领域专家。

## 🚀 这是什么？

The Agency 诞生于一个 Reddit 帖子，经过数月的迭代，成为一个精心打造的 AI agent 个性集合。每个 agent：

- 🎯 **专精**：在其领域拥有深厚专业知识（不是通用的提示词模板）
- 🧠 **个性驱动**：独特的语气、沟通风格和工作方法
- 📋 **交付导向**：产出实际的代码、流程和可衡量的结果
- ✅ **生产就绪**：经过实战检验的工作流和成功指标

你可以这样理解：你正在组建一支梦幻团队，只不过他们是不眠不休、从不抱怨、总能交付的 AI 专家。

## ⚡ 快速开始

### 选项 1：与 Claude Code 配合使用（推荐）
```bash
# 将所有 agents 安装到你的 Claude Code 目录
./scripts/install.sh --tool claude-code

# 如果只需要某个部门的 agents，可以手动复制
cp engineering/*.md ~/.claude/agents/

# 然后在 Claude Code 会话中激活任何 agent：
# “Hey Claude，激活前端开发者模式，帮我构建一个 React 组件。”
```

### 选项 2：作为参考文档使用

每个 agent 文件包含：
- 身份与性格特征
- 核心使命与工作流
- 附带代码示例的技术交付物
- 成功指标与沟通风格

浏览下方的 agents，然后复制或按需调整即可！

### 选项 3：与其他工具配合使用（GitHub Copilot、Antigravity、Gemini CLI、OpenCode、OpenClaw、Cursor、Aider、Windsurf、Kimi Code）
```bash
# 第一步：为所有支持的工具生成集成文件
./scripts/convert.sh

# 第二步：交互式安装（自动检测已安装的工具）
./scripts/install.sh

# 或者直接指定目标工具
./scripts/install.sh --tool antigravity
./scripts/install.sh --tool gemini-cli
./scripts/install.sh --tool opencode
./scripts/install.sh --tool copilot
./scripts/install.sh --tool openclaw
./scripts/install.sh --tool cursor
./scripts/install.sh --tool aider
./scripts/install.sh --tool windsurf
./scripts/install.sh --tool kimi
```
有关完整细节，请参阅下面的“多工具集成”部分。

## 📖 Agent 设计理念

每个 agent 的设计都包含：
- 🎭 **鲜明的个性**：不是通用模板，而是真实的人设和声音
- 📋 **清晰的交付物**：具体的产出物，而非模糊的指导
- ✅ **成功指标**：可衡量的结果和质量标准
- 🔄 **经过验证的工作流**：一步一步能奏效的流程
- 💡 **学习记忆**：模式识别和持续改进

## 🎁 是什么让它如此特别？

### 与通用 AI 提示词不同：
- ❌ 通用的“扮演一个开发者”的提示词
- ✅ 带有性格和流程的深度专精

### 与提示词库不同：
- ❌ 一次性的提示词集合
- ✅ 带有工作流和交付物的、全面的 agent 系统

### 与其他 AI 工具不同：
- ❌ 无法自定义的黑盒工具
- ✅ 透明、可复刻、可适应的 agent 个性

## 🎨 Agent 个性摘录

> “我不仅测试你的代码——我默认要找到 3-5 个问题，并且要求提供每个问题的视觉证据。”
> —— 证据收集者（测试部）

> “你不是在 Reddit 上营销——你是成为了一个碰巧代表品牌的、有价值的社区成员。”
> —— Reddit 社群建设者（市场部）

> “每个有趣的元素都必须服务于功能或情感目的。设计愉悦感，使其增强而非分散注意力。”
> —— 趣味注入师（设计部）
