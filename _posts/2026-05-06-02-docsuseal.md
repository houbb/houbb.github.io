# DocuSeal

DocuSeal 是一个开源平台，提供安全高效的数字文档签署与处理。

您可以在任何设备上通过易于使用、针对移动端优化的网页工具，创建 PDF 表单，供用户在线填写和签署。

[![演示视频](https://github.com/docusealco/docuseal/assets/5418788/d8703ea3-361a-423f-8bfe-eff1bd9dbe14)](https://demo.docuseal.tech)

## 功能特性
- PDF 表单字段构建器（所见即所得）
- 支持 12 种字段类型（签名、日期、文件、复选框等）
- 每个文档支持多位提交者
- 通过 SMTP 自动发送邮件
- 文件存储：本地磁盘或 AWS S3、Google Storage、Azure Cloud
- 自动 PDF 电子签名
- PDF 签名验证
- 用户管理
- 针对移动端优化
- 7 种 UI 语言，签署时支持 14 种语言
- 用于集成的 API 和 Webhook
- 数分钟内即可轻松部署

## 专业版功能
- 公司 Logo 与白标
- 用户角色
- 自动提醒
- 通过短信发送邀请及身份验证
- 条件字段与公式
- 通过导入 CSV、XLSX 电子表格批量发送

- 通过 HTML API 创建模板（[指南](https://www.docuseal.com/guides/create-pdf-document-fillable-form-with-html-api)）

#### Docker

```sh
docker run --name docuseal -p 3000:3000 -v.:/data docuseal/docuseal
```

默认情况下，DocuSeal Docker 容器使用 SQLite 数据库存储数据和配置。另外，也可以通过设置 `DATABASE_URL` 环境变量来使用 PostgreSQL 或 MySQL 数据库。

#### Docker Compose

下载 docker-compose.yml 到您的私有服务器：
```sh
curl https://raw.githubusercontent.com/docusealco/docuseal/master/docker-compose.yml > docker-compose.yml
```

通过 docker compose 在自定义域名下运行应用程序，并启用 https（请确保您的 DNS 指向服务器，以便 Caddy 自动签发 SSL 证书）：
```sh
sudo HOST=您的域名.com docker compose up
```
