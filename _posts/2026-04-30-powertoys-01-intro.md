## 📦 安装

有关详细的安装说明和系统要求，请访问[安装文档](https://learn.microsoft.com/windows/powertoys/install)。

若想快速开始，请选择以下安装方法之一：
<br/><br/>
<details open>
<summary><strong>从 GitHub 下载 .exe 文件</strong></summary>
<br/>

前往 [PowerToys GitHub 发行版](https://aka.ms/installPowerToys)，选择**资产**以显示安装文件，然后选择与您的体系结构和安装范围匹配的文件。对于大多数设备，应选择 _x64 每用户_。

<details>
<summary><strong>WinGet</strong></summary>
<br/>
通过 [WinGet](https://github.com/microsoft/winget-cli#installing-the-client) 下载 PowerToys。通过 winget 更新 PowerToys 时，将遵循当前的 PowerToys 安装范围。要安装 PowerToys，请从命令行 / PowerShell 运行以下命令：

- 用户范围安装程序（默认）

```powershell
winget install Microsoft.PowerToys -s winget
```

- 计算机范围安装程序

```powershell
winget install --scope machine Microsoft.PowerToys -s winget
```
</details>

## 贡献

这个项目欢迎所有类型的贡献。除了编写功能/修复 bug，其他协助方式还包括规范编写、设计、文档以及发现 bug。我们很高兴能与高级用户社区合作，打造一套工具，帮助您充分利用 Windows。我们恳请您**在开始处理您想要贡献的功能之前**，阅读我们的[贡献者指南](CONTRIBUTING.md)。我们将很乐意与您合作，以确定最佳方法，在整个功能开发过程中提供指导和辅导，并帮助避免任何浪费或重复的工作。大多数贡献都要求您签署一份[贡献者许可协议 (CLA)][oss-CLA]，声明您授予我们使用您贡献的权利，且您有权这样做。有关开发 PowerToys 的指导，请阅读[开发者文档](./doc/devdocs)以获取详细分解。这包括如何设置用于编译的计算机。
