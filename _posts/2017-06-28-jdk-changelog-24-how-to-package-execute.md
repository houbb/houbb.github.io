# exe4j

exe4j 是一个 Windows 启动器生成器，当你购买 install4j 时，它会免费提供。

exe4j 非常适合在不需要安装程序的情况下创建 Java 启动器。

所有 install4j 的许可证密钥都适用于 exe4j。

没有许可证密钥的情况下，exe4j 可以无限期地评估，但生成的可执行文件会显示评估警告，并且服务在 30 分钟后停止。
