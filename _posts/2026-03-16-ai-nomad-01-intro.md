# 推荐硬件配置

| 组件  | 推荐                              |
| --- | ------------------------------- |
| CPU | AMD Ryzen 7 / Intel Core i7 或更高 |
| 内存  | 32GB RAM                        |
| GPU | NVIDIA RTX 3060 或同级别 AMD GPU    |
| 存储  | 500GB 以上 SSD                    |

说明：

* 显存越大，可以运行越大的 AI 模型
* 推荐 Linux 系统

支持系统：

* Ubuntu
* Debian
* Windows（通过 Docker Desktop）

# 60 秒安装

在 Ubuntu 或 Debian 系统上运行：

```bash
curl -fsSL https://raw.githubusercontent.com/Crosstalk-Solutions/project-nomad/main/install/install_nomad.sh -o install_nomad.sh && sudo bash install_nomad.sh
```

要求：

* Ubuntu 22.04+
* Debian 12+

安装过程中：

* Docker 会自动安装
* 所需组件会自动部署
