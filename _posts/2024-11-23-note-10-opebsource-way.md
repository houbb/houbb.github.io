# 前言

个人感觉比较实用的发展流程

```

|
封装为 SDK 工具
|
使用 sdk，封装为 server 服务
|
多个平台，整合为平台
|
平台配合成为产品
|
产品解决实际的问题
```

我们可以在任何一个环节感兴趣处理，也可以跳过基础的，在基础的上面，构建自己的工具，解决实际的问题。
