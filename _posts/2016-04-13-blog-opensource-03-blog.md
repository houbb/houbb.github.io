# 开源地址

https://github.com/MQPearth/Blog

## Blog

#### 前端

* Vue(Vuex+Vue router)

* Element-UI

#### 后端

* SpringBoot
* MyBatis
* Spring security
* JWT
* Mongo
* Redis
* RabbitMQ
* Swagger-UI
* [ip2region](https://github.com/lionsoul2014/ip2region)

#### 项目目录

```
* blog-springboot              后端工程
* blog-vue                           前端工程
* img                                    文档相关图片
* other                                 其他文档
  * prototype.md		   原型文档
  * 博客缓存.png            redis博客缓存说明
  * 后端功能说明.png     后端功能说明
* conf                                   已配置的项目压缩文件
  * nginx.conf                 nginx配置文件，已配置好
* blog.sql                             mysql数据库结构
```
