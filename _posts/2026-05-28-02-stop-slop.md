# Stop Slop

一个剔除AI写作味儿的技能包。

## 这是什么

AI写作有套路——可预测的短语、僵化的结构、单一的节奏。这个技能包就是教Claude（或其他任何大语言模型）识别并清除这些痕迹的。

## 技能包结构

```
stop-slop/
├── SKILL.md           # 核心指令
├── references/
│   ├── phrases.md     # 需删除的短语
│   ├── structures.md  # 需避免的结构模式
│   └── examples.md    # 修改前后对照示例
├── README.md
└── LICENSE
```


## 快速开始

- **Claude Code**：将此文件夹添加为技能。
- **Claude Projects**：将 `SKILL.md` 和参考文件上传至项目知识库。
- **自定义指令**：从 `SKILL.md` 复制核心规则。
- **API调用**：将 `SKILL.md` 加入系统提示词，参考文件按需加载。


## 它能揪出什么

**禁用短语**
- 清嗓子式开场白、表强调的废话、商业行话、所有副词、空洞的陈述、元评论。

**结构陈词滥调**
- 二元对立、反面铺陈、故作破碎的短句、设问套路、假性拟人、旁观者视角、被动语态。

**句式规则**
- 禁止以特殊疑问词开头、禁止使用破折号、禁止滥用短句堆砌、禁止极端化表述、必须使用主动语态。


## 评分机制

在每个维度上按1-10分进行评分：

| 维度 | 问题 |
|---|---|
| 直接 | 是在陈述观点，还是只在做预告？ |
| 节奏 | 句式富于变化，还是像节拍器一样单调？ |
| 可信 | 尊重读者智商，还是居高临下？ |
| 真实 | 像人话，还是像AI语？ |
| 密度 | 有任何可以删掉的废话吗？ |

总分低于35分（满分50）——打回去重写。
