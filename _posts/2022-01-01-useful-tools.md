# git 设置

## 查看

```
git config user.name
git config user.email
git config user.password
```

## 修改

```
修改用户名
$ git config --global user.name "your name"

修改邮箱
$ git config --global user.email "your email@qq.com"

修改密码
$ git config --global user.password "xxx"
```
